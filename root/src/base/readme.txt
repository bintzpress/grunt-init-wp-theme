= How to install {%= title %} =

The theme is currently distributed as a zip file.

The process is below:

1. Obtain the zip file 
2. Log into your sites administration page
3. Click "Appearance" and then "Themes" from the side menu
4. Click the "Add New" button
5. Click the "Browse" button and select the zip file
6. Click the "Install Now" button and the theme will be uploaded
7. Click the "Activate" button to activate the theme
